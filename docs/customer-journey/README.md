## Comment lire chaque `.docx`

Chaque fiche contient :

1. **Objectif du parcours** — ce que le persona cherche à accomplir.
2. **Scénario (storyline)** — le récit en quelques phrases.
3. **La carte du parcours** — tableau : une colonne par étape, et pour chaque étape :
   objectif, actions, écran concerné, pensées/verbatim, émotion, points de douleur,
   opportunités produit, et **couverture produit** (déjà dans le site / à construire).
4. **Courbe émotionnelle** — satisfaction ressentie étape par étape (1 à 5).
5. **Synthèse** — moments de vérité et priorités produit.
6. **Traçabilité** — liens vers persona, story map, backlog, perturbation J1.
